# Лонгрид — статический шаблон

Коротко: это статическая страница на HTML/CSS/JS с адаптивной типографикой, оглавлением (TOC), прогресс-баром чтения и переключателем темы (светлая/тёмная).

Запуск локально:

1. Откройте папку проекта в терминале.
2. Запустите любой простой HTTP-сервер, например (в Linux):

```bash
python3 -m http.server 8000
```

3. Откройте http://localhost:8000/longread-site/

Деплой: можно загрузить содержимое папки `longread-site` в GitHub Pages (или на любой static hosting: Netlify, Vercel).

Автодеплой: в проект добавлён пример GitHub Actions workflow (`.github/workflows/deploy.yml`), который публикует содержимое репозитория в ветку `gh-pages` при push в `main`.

Структура:
- `index.html` — основной шаблон;
- `styles.css` — стили;
- `script.js` — скрипт: TOC/progress/theme/font-size/reveal + Chart.js init; навигация по главам осуществляется через боковую панель (TOC) — каждый пункт открывает свою главу;
- `assets/` — картинки и другие медиа (SVG-заглушки — замените на реальные PNG/JPG/WebP при необходимости);
